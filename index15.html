LDD (Une Langage de définition de données)

Créer une DB

CREATE DATABASE nom

Exemple :

CREATE DATABASE Magasin

supprimer une DB

DROP DATABASE nom

Exemple :

DROP DATABASE Magasin

Créer une Table

CREATE TABLE nom-table(
colonne1 type [contrainte],
colonne2 type [contrainte],
...
[CONSTRAINT] [nom] [contrainte],…)

Type de données Description
INT Entier
DECIMAL Réel
CHAR Chaîne de caractères de longueur fixe.
VARCHAR Chaîne de caractères de longueur variable dont la longueur maximale fixée.
TEXT Chaîne de caractères de longueur variable.
DATE Date.
TIME Temps.
DATETIME Date et Heure.

Contrainte

Contrainte Description
NOT NULL Garantit qu'une colonne ne peut pas avoir la valeur NULL.
DEFAULT Fournit une valeur par défaut pour une colonne lorsqu'aucune n'est spécifiée.
UNIQUE Garantit que toutes les valeurs d'une colonne sont différentes.
PRIMARY KEY Identifie de manière unique chaque ligne/enregistrement dans une table de BD.
CHECK Garantit que toutes les valeurs d'une colonne satisfont à certaines conditions.
Exemples des tables :
Ajouter une colonne à une table

ALTER TABLE table1
ADD colonne type [contrainte];

Supprimer une colonne à une table

ALTER TABLE table1
DROP colonne type [contrainte];

Exemple: ALTER TABLE Article
DROP poids DECIMAL ;

Modifier le type d’une colonne

ALTER TABLE table1
MODIFY colonne type [contrainte];

Exemple: ALTER TABLE Fournisseur
MODIFY NOM_fr VARCHAR(30) ;

Changer le type d’une colonne

ALTER TABLE table1
CHANGE nom_1 nom_2 type [contrainte];

Exemple: ALTER TABLE Fournisseur
CHANGE NOM_fr Libelle_Fr VARCHAR(10) ;

Ajouter une contrainte sur une colonne d’une table

ALTER TABLE table1
ADD CONSTRAINT nom [contrainte];

Exemple: ALTER TABLE Article
ADD CONSTRAINT NN1 CHECK (POIDS>0);

Supprimer une contrainte

ALTER TABLE table1
DROP CONSTRAINT nom;

Exemple: ALTER TABLE Article
DROP CONSTRAINT NN1;
