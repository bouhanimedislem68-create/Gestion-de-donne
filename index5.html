Table et données

  • Les données sont enregistrées dans des tables à deux dimensions.
  • Chaque table contient les données d’un sujet précis, appelée aussi entité.
  • Ligne = un enregistrement (tuple ou n-uplet).
  • Colonne = un attribut, propriété, caractéristique
  • Les tables sont reliées par des Relations
Pour chaque colonne ou attribut on définit:
  • Nom
  • Type
  • Taille maximale
  • Obligatoir/non obligatoire
  • Domaine
