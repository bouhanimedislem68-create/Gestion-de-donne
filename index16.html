LMD (Langage de manipulation de données)

Sélection de données

SELECT [DISTINCT] expression [,col,..ou*] FROM nom_tab [[AS] alias] [,nom_tab2]
[WHERE condition]
[GROUP BY critère]
[HAVING condition]
[ORDER BY [ASK/DESC] ]

Fonction Description Exemple Résultat
WHERE Filtre les lignes selon une condition SELECT * FROM employés WHERE salaire > 3000; Liste des employés avec un salaire > 3000
GROUP BY Regroupe les lignes ayant une même valeur dans une colonne SELECT département, COUNT(*) FROM employés GROUP BY département; Nombre d’employés par département
ORDER BY Trie les résultats selon une ou plusieurs colonnes SELECT * FROM employés ORDER BY salaire DESC; Liste des employés triée par salaire décroissant
HAVING Filtre les groupes après un GROUP BY SELECT département, AVG(salaire) FROM employés GROUP BY département HAVING AVG(salaire) > 4000; Départements avec un salaire moyen > 4000
Exemple :

SELECT nomElev, PrenomElev FROM élève WHERE idElev='123';

Insertion des données dans une table

INSERT INTO nom_table [(liste colonnes,...)] VALUES (liste valeurs,..)

Exemple :

INSERT INTO élève(idElev, nomElev, PrenomElev) VALUES ('923', 'Samir', 'Lousif');

Modification des valeurs d’une colonne

UPDATE nom_table SET colonne = expression
[WHERE condition]

Exemple :

UPDATE élève SET nomElev = 'Bouheni' WHERE idElev='123'

Suppression des enregistrements d’une table

DELETE FROM nom_table
[WHERE condition]

Exemple :

DELETE FROM élève WHERE idElev='345'

Les Fonctions d'Agregation

Fonction Description Exemple Résultat
COUNT() Compte le nombre de lignes SELECT COUNT(*) FROM employés; nombre des lignes
SUM() Calcule la somme d’une colonne numérique SELECT SUM(salaire) FROM employés; Somme des salaires
AVG() Renvoie la moyenne d’une colonne numérique SELECT AVG(salaire) FROM employés; moyenne des salaires
MAX() Renvoie la valeur maximale SELECT MAX(salaire) FROM employés; salaire maximale
MIN() Renvoie la valeur minimale SELECT MIN(salaire) FROM employés; salaire minimale

Les Fonctions de Type DATE

Fonction Description Exemple Résultat
DAY() Renvoie le jour du mois (1 à 31) SELECT DAY('2025-10-14'); 14
MONTH() Renvoie le mois (1 à 12) SELECT MONTH('2025-10-14'); 10
YEAR() Renvoie l’année SELECT YEAR('2025-10-14'); 2025
NOW() Renvoie la date et l’heure actuelles SELECT NOW(); 2025-10-14 12:51:00
CURDATE() Renvoie la date actuelle (sans l’heure) SELECT CURDATE(); 2025-10-14
ADDDATE() Ajoute un nombre de jours à une date SELECT ADDDATE('2025-10-14', 10); 2025-10-24
ADDDATE() avec INTERVAL Ajoute un intervalle spécifique (jours, mois, années...) SELECT ADDDATE('2025-10-14', INTERVAL 2 MONTH); 2025-12-14
DATEDIFF() Calcule la différence en jours entre deux dates SELECT DATEDIFF('2025-10-14', '2025-09-30'); 14
Jointure

En SQL , une Jointure est utilisée pour combiner des lignes provenat de deux ou plusieurs tables en fonction d'une condition qui lie les tables, souvent basée sur une relation entre une clé primaire et une clé étrangère.
Pour faire la jointure ; les tables sont listées dans la clause FROM , et la condition de jointure est définie dans la clause WHERE

Syntaxe:

SELECT Colonnes
FROM Table1 , Table2
WHERE Table1.Colonne = Table2.Colonne ;

On peut Aussi nommer les tables avec AS :

SELECT Colonnes
FROM Table1 AS A , Table2 AS B
WHERE A.Colonne = B.Colonne ;

Exemple :

Soient les tables : Clients et Commandes

Clients

Commandes

Requéte avec jointure :

SELECT A.Nom AS Client, C.Produit
FROM Clients AS A , Commandes AS C
WHERE A.ClientID = C.ClientID ;
