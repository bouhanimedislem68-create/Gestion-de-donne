La base de données

  • Une collection structurée de données, non redondantes, relatives à un ensemble de sujets et aux relations qui peuvent les concerner, enregistrée sur un support de stockage permanent .
  • Les données d’une base de données sont regroupées suivant leur appartenance à un sujet ou à une activité commune.
  • Appelé aussi Data Base "DB"
Exemple de BD : Netflix
